    Configure Call Home

    Call Home notification allows you to configure Spectre to automatically send an email alert to a support center to report system problems on specified devices (Fabric OS switches, routers, and directors).

    1. Click the Settings icon ( ), and then select Network Monitoring > Call Home.
    2. Click the name of Call Home center you want to configure.
    3. In the Call Home center drill-down page, click Add and Remove to add or remove devices. Click Configure to add or remove event filters from selected devices.
      Figure 11  Call Home drill-down page
    4. Select the Enable checkbox to enable the Call Home center.
    5. Click Save.
