    Configure AAA Services

    1. Click the Settings icon ( ), and then select General Settings > Authentication and Authorization.
    2. Select the primary authentication method, and then click Add to add an authentication server.
      Figure 18  Adding an authentication server
    3. Provide the server credentials in the dialog box.
    4. Click Save.
